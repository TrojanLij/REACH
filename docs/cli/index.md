# CLI reference

Top-level groups:
- `reach version`
- `reach server ...`
- `reach routes ...`
- `reach logs ...`
- `reach forge ...`
- `reach dns ...`
- `reach dev ...`

Run `--help` on any command for full option details.

## Version commands
Show runtime + component versions:
```bash
reach version
```

Show a single component:
```bash
reach version --component ui
```
